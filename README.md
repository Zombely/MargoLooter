# MargoLooter

## Description

Looter logger for showing item drops from monsters in game `margonem`.
Collecting user data from obtaining loot.

## Set up

`pip install -r requirements.txt`

## Main sites

- `https://margonem.pl`
- `http://emargo.pl`
- `https://margohelp.pl`

## Stack

- Python 3.10.9
- Javascript
